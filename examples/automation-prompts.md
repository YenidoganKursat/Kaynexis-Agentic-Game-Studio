# Automation / Recurring Task Prompt Ideas

Use these prompts as inspiration for Codex app automations or recurring manual checks. Review them manually before scheduling anything.

## Weekly health check
```text
$project-radar Scan the repo for the highest-impact gaps and update the risk register if new blockers appear.
```

## Build / release drift check
```text
$release-gate Review release readiness from current docs and recent changes. Flag blockers and missing evidence only.
```

## Bug inbox triage
```text
$bug-triage Convert the newest high-severity crash reports into reproducible reports with owners and next steps.
```

## LiveOps planning
```text
$liveops-calendar Update the next 4 weeks of beats based on current content capacity and risk notes.
```

## Community follow-up
```text
$community-ops Summarize player feedback themes from the latest playtest notes and propose 3 concrete follow-up actions.
```
