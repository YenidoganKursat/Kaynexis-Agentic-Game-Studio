# Kaynexis Agentic Game Studio

A compact Godot 4 action prototype centered on a single combat room.

## Current Slice

- Top-down movement
- Dash timing challenge against the Pulse Warden
- Three-hit fail state with room reset
- One reward choice after clearing the encounter

## Controls

- Move: `WASD` or arrow keys
- Dash: `Space` or `Enter`
- Reward choice: `1` or `2`

## Run

1. Open the folder in Godot 4.x
2. Load `project.godot`
3. Run the main scene

## Export

Included presets:

- `Linux/X11`
- `Windows Desktop`

Default outputs:

- `build/linux/kaynexis-agentic-game-studio.x86_64`
- `build/windows/kaynexis-agentic-game-studio.exe`
