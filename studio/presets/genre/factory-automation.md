# Genre Preset — Factory Automation

Inputs become outputs through player-built systems, throughput becomes the main puzzle, and scale exposes clarity, logistics, and performance problems.

## Default assumptions
- Systems should compose predictably
- Throughput and bottlenecks must be inspectable
- Data representation choices matter early
- Expansion increases both power and maintenance cost

## Must-watch risks
- Logistics becoming unreadable at scale
- Simulation tick cost growing faster than content value
- Blueprint or build tools arriving too late
- Production chains feeling like chores instead of insight
