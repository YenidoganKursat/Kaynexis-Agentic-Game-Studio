# Genre Preset — Hero Shooter

Role-defined heroes, objective-driven matches, teamfight readability, cooldown-based kits, and strong combat presentation.

## Default assumptions
- Each hero should own a clear role or fantasy
- Match objectives matter more than pure elimination count
- Combat readability and netcode trust are core features
- Balance and presentation will need continuous iteration

## Must-watch risks
- Heroes blurring together
- Objective modes becoming unreadable at speed
- Visual clutter hiding team and enemy intent
- Network or performance issues eroding combat trust
