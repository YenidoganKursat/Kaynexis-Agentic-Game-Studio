# Genre Preset — Soulslike

Stamina-driven combat, dangerous exploration, readable telegraphs, punitive but fair recovery, and shortcut-heavy world design.

## Default assumptions
- Every attack commitment has a cost
- Enemy telegraphs must be readable
- Death is part of the learning loop, not a failure of the design
- Checkpoints and shortcuts are core pacing tools

## Must-watch risks
- Telegraphs becoming too subtle
- Punishment feeling arbitrary instead of instructive
- Checkpoint structure failing to reduce repetition fatigue
- Enemy rhythm or recovery windows feeling inconsistent
