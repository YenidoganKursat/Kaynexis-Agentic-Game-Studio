# Genre Preset — Action Roguelite

Short runs, compounding build variety, readable combat, meaningful risk/reward, and strong reset value.

## Default assumptions
- 20–45 minute runs
- High encounter readability
- Meta progression without invalidating mastery

## Must-watch risks
- Run variety collapse
- Snowball balance
- Too much permanent power
- Content fatigue

- $studio-start
- $combat-loop
- $metaprogression-loop
- $qa-matrix
