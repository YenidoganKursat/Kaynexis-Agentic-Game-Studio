# Genre Preset — Extraction Lite

High-risk session stakes, secure economy, readable escape windows, and anti-cheat/exploit discipline.

## Default assumptions
- Session-to-meta loop
- PvE or PvPvE pressure
- Loadout risk/reward

## Must-watch risks
- Cheat/exploit economy abuse
- Death feels unfair
- Match pacing collapse

- $combat-loop
- $economy-balance
- $security-threat-model
