# Genre Preset — Survivorlike

High-density enemy pressure, automated or low-friction attacks, movement-first skill expression, and rapid build escalation over short runs.

## Default assumptions
- 10–30 minute runs
- Continuous enemy spawns with readable density
- Upgrade cadence every few minutes or faster
- Performance budget matters from the first prototype

## Must-watch risks
- Visual clutter overwhelming dodge readability
- Performance collapse under enemy and projectile count
- Build options feeling fake because one path dominates
- Surviving too long without meaningful pacing shifts

- $studio-start
- $combat-loop
- $perf-pass
