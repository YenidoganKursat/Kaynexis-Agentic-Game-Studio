# Genre Preset — Metroidvania

Exploration-first progression, gated traversal abilities, strong world memory, and satisfying backtracking where movement upgrades recontextualize old spaces.

## Default assumptions
- A connected world matters more than isolated levels
- Traversal upgrades reshape both navigation and combat
- Map readability and player memory are core UX surfaces
- Backtracking must feel rewarding, not wasteful

## Must-watch risks
- Gating becoming arbitrary instead of teachable
- Map or waypoint UX failing to support return play
- Ability unlock order breaking pacing
- Backtracking feeling empty after the first visit
