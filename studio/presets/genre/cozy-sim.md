# Genre Preset — Cozy Sim

Low-stress progression, high routine clarity, strong environmental attachment, and accessible UI.

## Default assumptions
- Session-friendly chores/goals
- Readable schedules and state changes
- Warm feedback loops

## Must-watch risks
- Excessive grind
- UI clutter
- Weak reward texture
- Tedious traversal
