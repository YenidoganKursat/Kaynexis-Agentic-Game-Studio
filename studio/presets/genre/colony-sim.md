# Genre Preset — Colony Sim

Autonomous agents, layered needs, production chains, crisis recovery, and long-horizon management where the player shapes systems more than direct avatar action.

## Default assumptions
- Many interacting simulation systems
- Agent priorities and jobs are more important than twitch control
- Readable information surfaces matter as much as simulation depth
- Failure and recovery are part of the intended loop

## Must-watch risks
- Simulation opacity making losses feel random
- UI overload hiding the most urgent colony problem
- Agent priority rules producing frustrating edge cases
- Save/load or long-session state drift
