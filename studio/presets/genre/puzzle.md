# Genre Preset — Puzzle

Elegant rule set, readable affordances, fair hinting, and fast iteration on player misunderstanding.

## Default assumptions
- Small mechanic set
- Clarity beats spectacle
- Hints protect flow without spoiling

## Must-watch risks
- Rule ambiguity
- Red herrings
- Hard locks
- Insufficient teach/test structure
