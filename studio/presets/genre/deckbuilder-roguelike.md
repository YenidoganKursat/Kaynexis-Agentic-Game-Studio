# Genre Preset — Deckbuilder Roguelike

Run-based card or deck construction, compounding synergies, route planning, readable combat states, and meaningful drafting tension.

## Default assumptions
- 20–60 minute runs
- Small but legible combat state space
- Drafting and route choice matter as much as moment-to-moment execution
- Run state resets cleanly while unlocks stay bounded

## Must-watch risks
- Dominant deck archetypes removing discovery
- Card text or combat state becoming unreadable
- Unlock bloat collapsing draft quality
- Encounters failing to pressure different deck shapes
