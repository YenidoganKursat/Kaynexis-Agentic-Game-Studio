# Platform Preset — Web Demo

Fast load, low friction, browser-safe input/audio behavior, and clear CTA toward wishlist or download.

## Operating assumptions
- Players drop fast if load is long
- Embed/browser quirks matter
- CTA must be explicit

## Must-watch risks
- Asset bloat
- Input focus bugs
- Audio autoplay limitations
- Weak CTA

## Recommended skills
- $web-build-readiness
- $storefront-launch
- $marketing-beat-brief
