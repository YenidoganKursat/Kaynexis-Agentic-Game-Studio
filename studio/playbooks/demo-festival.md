# Playbook — Demo / Festival Build

Optimize for a short, memorable, low-friction public demo with clear CTA.

## Focus
- Fast onboarding
- short content loop
- safe reset path
- store CTA

## Recommended skills
- $scope-rescue
- $tutorial-onboarding
- $web-build-readiness
- $marketing-beat-brief

## Exit criteria
- 2–10 minute clean loop
- stable reset
- wishlist/download CTA ready
