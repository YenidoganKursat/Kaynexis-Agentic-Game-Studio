# Playbook — Porting Program

Treat a port as a product adaptation, not a blind build export.

## Focus
- Input changes
- performance targets
- compliance
- cert evidence

## Recommended skills
- $porting-plan
- $mobile-optimization
- $console-cert-readiness
- $release-gate

## Exit criteria
- Platform delta map complete
- cert/perf blockers known
- owner plan agreed
