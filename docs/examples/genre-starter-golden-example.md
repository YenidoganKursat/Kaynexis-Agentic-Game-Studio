# Genre Starter Golden Example

## Selected preset

- Preset slug: `action-roguelite`
- Summary: A readable, timing-driven combat loop with upgrades between rooms.

## Default assumptions

- One-player run structure first
- Fast restart cadence
- Clear readability over content volume

## Must-watch risks

- Enemy clutter outgrows readability
- Upgrade choices become flat
- Encounter pacing loses pressure

## Suggested first slice

- One combat room
- One enemy role
- One upgrade choice
- One clear success/failure loop

## Reference games

- Hades
- Dead Cells
- Vampire Survivors

## Design focus

- Preserve a readable dominant loop before scaling content

## Advanced genre guidance

- Use the maturity framework once the first slice is stable and you need stronger support for balance, telemetry, accessibility, and content throughput.
